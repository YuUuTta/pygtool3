pygtool module¶

pygtool3.pygtool is a interface to pygtool3. It opens gtool3 formatted data which was developed by Numaguchi(1999) We have to know whether input data is 2D or 3D in advance.

file : string, filename count : int, num of data, default:1 x,y : int, num of each grid. default (x,y)=(128,64)

gtool2d : pygtool3.pygtool_core.Gtool2d

file : string, filename count : int, num of data, default:1 x,y,z : int, num of each grid. default (x,y,z)=(128,64,36)

default:None it reads GTAXDIR/GTAXLOC.GLON(x),GTAXLOC.GGLA(y)
